### 📱 2. Для Android (Мобильная версия)

> [!NOTE]
> Обычный мобильный Chrome не поддерживает расширения. Для работы YFull Finder на Android используйте браузеры с поддержкой расширений.

#### Рекомендуемый способ: Mozilla Firefox (Нативная поддержка)
**Mozilla Firefox** — наиболее стабильный вариант для Android с полной поддержкой расширений.

1. Установите **Firefox** из [Google Play Market](https://play.google.com/store/apps/details?id=org.mozilla.firefox).
2. Откройте Firefox и перейдите в **Меню** (три точки) → **Дополнения** (Add-ons).
3. Найдите **YFull Finder** через поиск или перейдите по [прямой ссылке](https://addons.mozilla.org/en-US/firefox/addon/yfull-finder-ajwla/).
4. Нажмите **«Добавить в Firefox»**.

> [!TIP]
> **Для удобства работы:** В настройках Firefox включите **«Режим для ПК»** (Desktop Mode) на сайте YFull.com. Это позволит видеть полное дерево без мобильных ограничений.  
> Для этого: Меню → Настройки → Сайт → Режим просмотра → **Версия для ПК**.

#### Альтернативный способ: Kiwi Browser
1. Установите **Kiwi Browser** из [Google Play Market](https://play.google.com/store/apps/details?id=com.kiwibrowser.browser).
2. В Kiwi Browser откройте страницу [Chrome Web Store](https://chromewebstore.google.com/detail/yfull-finder-ajwla/nbilonjoanaocbjbhepbbcdbgddlpepi).
3. Найдите **YFull Finder** и нажмите «Установить».

## 🚀 Часть 3. Использование функций

### 1. Автоматическое обогащение дерева

Как только вы заходите на сайт YFull.com, расширение автоматически распознает ID образцов и накладывает слой генеалогических данных.

*   **Было:** Только код (например, `YF143588`).
*   **Стало:** Цветная метка (чип) с фамилией, этносом, регионом и национальным флагом.

<img width="1244" height="444" alt="image" src="https://github.com/user-attachments/assets/a7fba7a2-2ba1-4984-a68f-cc29f6d4f48a" />


*   **🏴 30+ этнических флагов:** Поддерживаются флаги для кавказских, тюркских, уральских, балтийских и других этнических групп. Все SVG-флаги стандартизированы по единому соотношению сторон для аккуратного отображения.

*   **📱 Мобильные тултипы:** На сенсорных устройствах чипы с данными работают в режиме **tap-to-toggle** — нажмите на чип, чтобы увидеть подробную информацию. Повторное нажатие скроет тултип.

> [!NOTE]
> Расширение **не изменяет** научные данные YFull. Оно только добавляет визуальный слой с генеалогической информацией поверх существующего интерфейса.

### 2. Добавление данных (Contribution)

Вы можете помочь сообществу, добавив информацию об известном вам образце:

1. Найдите образец с серой иконкой **[+]** (это значит, данных еще нет).

3. Выберите тип записи (**Y-ДНК дерево** или **Мито-ДНК дерево**) в верхней части сайта.

4. Заполните поля. Форма поддерживает **каталог полей** — нажмите **[+ Добавить поле]** чтобы увидеть все доступные поля в визуальном каталоге:
   - **Surname** (Фамилия) — обязательное
   - **Tribe/Ethnicity** (Этнос/Племя) — обязательное
   - **Village** (Село)
   - **Region** (Исторический регион)

   - **Gorge** (Ущелье)
   - **Shakhar** (Шахар)
   - **Abipara** (Абипара)
   - **Krsna Slava** (🇷🇸 Сербская крёстная слава / Покровитель)
   - **Info** (Дополнительная информация)
   - **Link** (Ссылка на источник)
<img width="774" height="733" alt="image" src="https://github.com/user-attachments/assets/f0bba550-2576-474e-97fe-abc4ef0fcdda" />

### 3. Анонимный вклад (Shadow Profile)

Если вы хотите поделиться данными, но сохранить анонимность:

*   В форме добавления поставьте галочку **«Make Anonymous»**.
*   Ваш вклад будет сохранен, но в истории изменений он будет отмечен как **«Anonymous»**.
*   **Важно:** Анонимность скрывает только ваше имя как автора. Сами генеалогические данные (фамилия, этнос) остаются публичными для пользы всего сообщества.

> [!NOTE]
> Даже без регистрации ваши вклады сохраняются через систему **Shadow Profile** (теневого профиля). Если позже вы зарегистрируетесь, все ваши предыдущие вклады будут автоматически привязаны к вашему аккаунту.

### 6. Рейтинг (Leaderboard) 🏆

Рейтинг — это система признания заслуг наших самых активных волонтёров. Он отображает пользователей, которые добавили больше всего образцов в базу данных.

*   **Где найти:** Откройте вкладку **Rating** (Рейтинг) на веб-портале.
*   **Фильтрация:** Вы можете фильтровать список по:
    - **Типу ДНК:** Y-ДНК, mtDNA или Все вместе.
    - **Периоду:** 7 дней, 30 дней или За всё время.
*   **Ваша позиция:** Система автоматически подсвечивает ваше место в списке, даже если вы ещё не вошли в Топ-10.
*   **📈 Тренды позиций:** Рядом с каждым участником отображается индикатор изменения позиции (↑ рост, ↓ падение, — стабильно), как на фондовой бирже.
*   **🙈 Приватность:** Если вы хотите скрыть своё имя из рейтинга, активируйте опцию **«Скрыть в рейтинге»** в настройках профиля. Ваши образцы всё равно будут учитываться, но имя отобразится как «Anonymous».

### 7. Массовая загрузка (Bulk Upload) 📦

Для опытных пользователей: добавляйте сразу **несколько образцов** одним действием через JSON-формат.

1. Откройте портал [**yfinder.apsny.dev**](https://yfinder.apsny.dev) и авторизуйтесь.
2. Нажмите кнопку **«Bulk Upload»** (Массовая загрузка).
3. В открывшемся окне:
   - Выберите **целевую базу данных** (Main / AADNA) и **тип ДНК** (Y-DNA / mtDNA).
   - Вставьте данные в формате JSON:

```json
[
  {
    "id": "YF123456",
    "surname": "Ivanov",
    "tribe": "Russian",
    "village": "Moscow"
  },
  {
    "id": "YF789012",
    "surname": "Petrov",
    "tribe": "Ukrainian"
  }
]
```

4. **JSON-помощник:** 🛠 Нажмите кнопку **«Code Helper»** для визуального конструктора JSON:
   - Просмотрите **каталог всех доступных полей** с описаниями
   - Отметьте нужные поля галочками
   - Сгенерируйте **шаблон JSON** одним кликом
   - Скопируйте шаблон в буфер обмена или вставьте прямо в форму
<img width="881" height="690" alt="image" src="https://github.com/user-attachments/assets/c8e45e74-eb6a-4983-8a75-86cbfd9e9cf1" />


5. Система автоматически **проверит дубликаты** и покажет, какие ID уже существуют в базе.
6. Нажмите **Upload** для отправки. Запрос будет создан и направлен на модерацию.

> [!NOTE]
> Массовая загрузка доступна только для **авторизованных пользователей**. Запрос проходит модерацию перед добавлением в базу.

**Способ 1: Плавающая кнопка**
1.  Найдите **иконку камеры** 📷 в правом нижнем углу любой страницы дерева.
2.  Нажмите ее, чтобы открыть инструмент создания скриншотов.

**Способ 2: Попап расширения**
1.  Нажмите на иконку расширения в панели браузера.
2.  Выберите пункт **«Скриншот»** в меню.

**Режимы захвата:**
*   **Выделение области:** Нарисуйте прямоугольник мышкой для захвата нужной части дерева. Интерфейс выделения выполнен в стиле **Ubuntu 24.04 LTS** — тёмная область вокруг с размерами выделения.

*   **Полное дерево:** Захват всего дерева целиком, включая вложенные ветви.
*   **Научное дерево (Scientific View):** Полный захват с таймлайном.
*   **Диаграмма (Chart View):** Захват с раскрытием вложенных структур.

**Форматы экспорта:**
*   **PNG** — растровое изображение высокого качества.
*   **SVG** — векторный формат для масштабирования без потери качества.
*   **PDF** — документ для печати и обмена.

**Дополнительные возможности:**
*   **📋 Автокопирование в буфер обмена:** После создания скриншота изображение автоматически копируется — просто вставьте (Ctrl+V) в чат или документ.
*   **🔖 Водяной знак проекта:** Логотип проекта автоматически добавляется в угол скриншота. Его позиция адаптируется, чтобы не закрывать важные данные. Предпросмотр водяного знака виден уже во время выделения области.
*   **🧹 Чистый захват:** Лишние элементы интерфейса (кнопки, виджеты) автоматически скрываются перед захватом.

### 11. Аннотации на скриншотах ✏️

После выделения области открывается **продвинутый редактор аннотаций**, позволяющий добавить надписи и пометки прямо на скриншот:

**Текстовые аннотации:**
*   Нажмите на скриншот, чтобы добавить текстовую надпись.
*   Откроется **контекстное меню** с настройками прямо рядом с курсором:
    - **Шрифт:** Выбор из стандартных (Inter, Montserrat, Lobster) и **аутентичных абхазских** шрифтов (CCarial, LiteraturAB).
    - **Размер шрифта**
    - **Стиль:** Жирный, Курсив, Подчёркнутый, Зачёркнутый
    - **Прозрачность текста**
    - **Цвет текста:** Палитра цветов
    - **Цвет фона:** Палитра цветов
    - **Контур текста:** Палитра цветов (используется технология 16-точечного кругового text-shadow для чётких контуров)

*   **Перемещение:** Иконка перетаскивания (`⋯`) позволяет перемещать текстовый блок перед финализацией.
*   **Отмена (Ctrl+Z):** Быстрый откат последнего действия.
*   **Маска области:** Все рисунки и тексты строго ограничены выделенной областью скриншота.

### 12. Транслитерация (Кириллица → Латиница) 🔤

Для удобства международного сообщества предусмотрена **транслитерация одним кликом:**

*   При вводе кириллического текста в поля (Фамилия, Этнос, Село и др.) появляется предупреждение с кнопкой **«Транслитерировать»**.
*   Нажмите на кнопку — текст автоматически конвертируется в латиницу: `Иванов` → `Ivanov`.
*   Работает во всех формах: добавление, редактирование и модальные окна.

> [!TIP]
> Для удобства всех пользователей рекомендуется вводить данные **латинскими буквами**. Транслитерация помогает быстро конвертировать уже введённый кириллический текст.
